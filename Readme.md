## Requirements
- Visual Studio 2022 with ASP.NET CORE Packages
- .NET 8.0 SDK
- Node JS 
- NPM package manager for installing Tailwind CSS

## Installation and Building
```shell
git clone https://github.com/kisenaa/FinalExam_pweb
cd FinalExam_pweb
dotnet build
dotnet run
```

## Publishing 
```shell
cd FinalExam_pweb
dotnet build
dotnet publish
dotnet serve --port 8080
```
